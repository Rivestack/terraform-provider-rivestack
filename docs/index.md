# Rivestack Provider

The Rivestack provider allows you to manage [Rivestack](https://rivestack.io) HA PostgreSQL clusters and their configuration using Terraform.

## Authentication

The provider authenticates using a Rivestack API key. You can generate an API key from the Rivestack dashboard.

Set the `RIVESTACK_API_KEY` environment variable:

```bash
export RIVESTACK_API_KEY="rsk_your_api_key_here"
```

Or configure it directly in the provider block:

```hcl
provider "rivestack" {
  api_key = var.rivestack_api_key
}
```

## Example Usage

```terraform
terraform {
  required_providers {
    rivestack = {
      source  = "rivestack/rivestack"
      version = "~> 0.1"
    }
  }
}

provider "rivestack" {
  api_key = var.rivestack_api_key
  # base_url = "https://api.rivestack.io" # optional, this is the default
}

variable "rivestack_api_key" {
  type      = string
  sensitive = true
}
```
